# Быстрая установка и запуск

## Вариант 1: Автоматическая установка (рекомендуется)

```bash
# Дайте права на выполнение скрипта (если нужно)
chmod +x setup.sh run.sh

# Запустите установку
./setup.sh

# Запустите приложение
./run.sh
```

## Вариант 2: Ручная установка

### 1. Клонирование и переход в папку проекта
```bash
cd commercial_proposal_analyzer
```

### 2. Создание виртуального окружения
```bash
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка окружения
```bash
cp .env.example .env
# Отредактируйте .env файл и выберите провайдер LLM
```

### 5. Создание тестовых данных (опционально)
```bash
python utils/create_test_data.py
```

### 6. Запуск приложения
```bash
streamlit run main.py
```

## Настройка LLM

### Вариант A: OpenRouter (облако)

1. Зарегистрируйтесь на [OpenRouter](https://openrouter.ai/)
2. Перейдите в настройки аккаунта → API Keys
3. Создайте новый API ключ
4. Скопируйте ключ в файл `.env`:
   ```
   LLM_PROVIDER=openrouter
   OPENROUTER_API_KEY=ваш_ключ_здесь
   ```

### Вариант B: Ollama (локально)

1. Установите Ollama: [https://ollama.com](https://ollama.com)
2. Запустите сервис:
   ```bash
   ollama serve
   ```
3. Скачайте модель (пример):
   ```bash
   ollama pull llama3.1
   ```
4. Укажите в `.env`:
   ```
   LLM_PROVIDER=ollama
   OLLAMA_BASE_URL=http://localhost:11434/v1
   OLLAMA_MODEL=llama3.1
   ```

## Тестирование без API ключа

Приложение поддерживает два режима анализа:
- **Простой анализ** - работает без API ключа, выбирает лучшее предложение по цене и скидке
- **LLM анализ** - требует API ключ, выполняет комплексную оценку с объяснением выбора

Вы можете использовать простой анализ для тестирования функциональности.
