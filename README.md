# Анализатор коммерческих предложений

Веб-приложение на Python для анализа и выбора лучшего коммерческого предложения с использованием LLM (OpenRouter или локальная Ollama) и Streamlit UI.

## Функциональность

- 📁 Загрузка Excel файлов с коммерческими предложениями
- 🤖 Интеллектуальный анализ с помощью LLM (OpenRouter или локальная Ollama)
- 🤖 Telegram-бот для анализа Excel файлов в чате
- 📊 Автоматический выбор лучшего предложения
- 📝 Подробное объяснение выбора с анализом сильных/слабых сторон
- ⚙️ Настройка весов критериев оценки
- 📤 Экспорт результатов в JSON/CSV

## Требования к данным

Excel файл должен содержать следующие колонки:
- **Контрагент** - название компании-поставщика
- **товар** - наименование товара или услуги
- **цена** - цена в числовом формате
- **скидка** - скидка в процентах
- **условия поставки** - текст с условиями доставки и оплаты

## Установка и запуск

### 1. Клонирование и настройка

```bash
git clone <repository_url>
cd commercial_proposal_analyzer
```

### 2. Создание виртуального окружения (рекомендуется)

```bash
python -m venv venv
source venv/bin/activate  # для Linux/Mac
# или
venv\Scripts\activate  # для Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка LLM (OpenRouter или Ollama)

1. Создайте файл `.env` в корне проекта:
```bash
cp .env.example .env
```
2. Выберите один из вариантов:

- **OpenRouter (облако)**:
```
LLM_PROVIDER=openrouter
OPENROUTER_API_KEY=your_api_key_here
```

- **Ollama (локально)**:
```
LLM_PROVIDER=ollama
OLLAMA_BASE_URL=http://localhost:11434/v1
OLLAMA_MODEL=llama3.1
```
Убедитесь, что локальный сервер запущен (`ollama serve`) и модель загружена (`ollama pull llama3.1`).

### 5. Запуск приложения

```bash
streamlit run main.py
```

Приложение будет доступно по адресу: `http://localhost:8501`

### 6. Запуск Telegram-бота

1. Создайте бота через `@BotFather` и получите токен.
2. Добавьте токен в `.env`:
```
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
```
3. (Опционально) Настройте режим анализа и веса:
```
TELEGRAM_USE_LLM=true
TELEGRAM_WEIGHT_PRICE=0.4
TELEGRAM_WEIGHT_DELIVERY=0.3
TELEGRAM_WEIGHT_RELIABILITY=0.3
```
4. Запустите бота:
```bash
python telegram_bot.py
```

## Использование

1. **Загрузите Excel файл** с коммерческими предложениями
2. **Настройте критерии** оценки в боковой панели
3. **Запустите анализ** - выберите тип анализа (простой или LLM)
4. **Просмотрите результаты** - лучшее предложение, объяснение, альтернативы
5. **Экспортируйте результаты** в JSON или CSV формате

## Типы анализа

### 1. Простой анализ
- Выбор по минимальной цене со скидкой
- Выбор по максимальной скидке
- Быстрая обработка без использования LLM

### 2. LLM анализ (рекомендуется)
- Комплексная оценка по нескольким критериям
- Учет условий поставки и надежности контрагента
- Подробное объяснение выбора
- Анализ рисков и рекомендации
- Поддержка провайдеров OpenRouter и Ollama

## Критерии оценки в LLM анализе

1. **Цена** (с учетом скидки) - 40% по умолчанию
2. **Условия поставки** - сроки, надежность, гибкость - 30%
3. **Надежность контрагента** - репутация, опыт - 30%
4. **Качество товара/услуги**
5. **Общая выгода для компании**

Веса критериев можно настраивать в интерфейсе.
